READMEConnect Four GameHow to play:* The aim of the game is to connect four of your coloured counters in a row horizontally, vertically or diagonally.* Each player will take turns placing a counter into the connect four board by typing a number between 1 and 7 into the command line and then pressing enter * Once one player successfully connects four counters in a row, the winner will be specified and the game will quit* If the board becomes full and neither player has won then the players have drawn.Special Features:* If a player picks a column that is full, the player will be prompted to pick another column which as an empty space available.
